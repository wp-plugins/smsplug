With this plugin you can use the service from aspsms.com. It allows you to send sms via their service.

== Description ==

This is a plugin / widget, after installing you have the possibility to locate a widget to your site.
An visitor can now send a sms to a person. 

== Installation ==

1. Upload the folder "SMSPlug" to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "Settings" in the admin pannel and set the UserID, Password and the Originator. Maybe you wan't to display something different than "SMSPlug" on the widget.

== Frequently Asked Questions ==

= Do I have to pay for the SMS? =

Yes, you can buy these on smsblaster.ch

= Where do I find the UserID? =

The Userkey can be found on smsblaster.ch it is not your email adress
